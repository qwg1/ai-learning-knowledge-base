# 🛡️ 操作前强制检查清单

> 每次操作前必须检查，避免重复犯错

---

## 🚨 操作前必须问自己

```
1. 做这件事需要什么技能？
2. 知识库有没有相关经验？
3. 以前有没有犯过类似错误？
4. 有没有操作指南或复盘？
5. 需要先读哪些文件？
```

---

## 📋 检查流程

### Step 1: 回忆（3秒）
```
Q: 我以前做过类似的事吗？
A: 有 → 去知识库找经验
A: 没有 → 先搜索/学习再操作
```

### Step 2: 搜索知识库（10秒）
```
必须搜索：
1. MEMORY.md（长期记忆）
2. 今日memory/（短期记录）
3. 相关技能文档（如GIT_SETUP_GUIDE.md）
4. 错误复盘（如GIT_RETROSPECT.md）
```

### Step 3: 执行操作
```
根据找到的经验执行
不要凭记忆操作
```

### Step 4: 记录结果
```
成功 → 总结经验，更新知识库
失败 → 记录错误，写复盘
```
