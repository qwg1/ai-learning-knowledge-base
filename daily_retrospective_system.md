# 每日复盘学习机制

## 目标
每天结束时总结问题、复盘学习、持续优化

## 执行时间
每天会话结束前 或 重要任务完成后

## 复盘模板

### 1. 今日完成事项
- 列出完成的任务

### 2. 遇到的问题
- 问题描述
- 根本原因分析

### 3. 解决方案
- 已采用的解决方法
- 效果评估

### 4. 学到的经验
- 技术要点
- 方法论总结

### 5. 待改进项
- 流程优化
- 工具改进

### 6. 明日计划
- 继续推进的任务
- 预判可能遇到的问题

## 知识沉淀
- 更新 MEMORY.md
- 更新 daily memory
- 推送到GitHub知识库

## 自动化工具
- 创建复盘脚本
- 定时提醒机制
